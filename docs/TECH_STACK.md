# Tech Stack Documentation

## Frontend Framework

**Next.js 14**
- React-based framework
- Built-in routing (file-based)
- Server-side rendering (SSR) and static generation
- API routes built-in
- Performance optimized
- Vercel integration (maker of Next.js)

## Styling

**Tailwind CSS**
- Utility-first CSS framework
- No custom CSS needed (mostly)
- Responsive design helpers
- Dark mode support
- Easy to customize

## Form Handling

**Next.js Server Actions** or **React Hook Form**
- Server Actions: Built into Next.js, no extra library needed
- React Hook Form: Lightweight form library if needed

## Image Optimization

**Next.js Image Component**
- Automatic optimization
- Responsive images
- Lazy loading
- WebP conversion

## Contact Form Backend

**Nodemailer** or **SendGrid** or **Formspree**
- Send emails from contact form
- Keep it simple for portfolio

## Hosting & Deployment

**Vercel**
- Free tier available
- Automatic deployments from GitHub
- Built by creators of Next.js
- Custom domains supported
- Environment variables built-in

## Development Tools

- **Node.js** - JavaScript runtime
- **npm/yarn/pnpm** - Package manager
- **Git** - Version control
- **VS Code** - Code editor

## Optional Enhancements

- **Framer Motion** - Animations
- **Sharp** - Image processing
- **TypeScript** - Type safety (can add later)

---

## Installation Command (Next.js)

```bash
npx create-next-app@latest
```

This will prompt you to choose options:
- ✅ TypeScript (optional, can say no)
- ✅ ESLint
- ✅ Tailwind CSS
- ✅ App Router (newer Next.js routing)
